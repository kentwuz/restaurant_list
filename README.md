# RESTAURANT LIST

專案使用Node.js與Express/MongoDB打造的餐廳美食網站，提供瀏覽不同風格的餐廳以及其相關資訊; 包括名字/聯繫方式/地點/照片以及評分，同時也提供新增餐廳以及刪除餐廳資料的功能。

## Installation

```git clone https://github.com/kentwuz/restaurant_list.git
cd restaurant_list
npm install
```
